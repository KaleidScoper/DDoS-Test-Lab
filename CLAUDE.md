# DDoS-Test-Lab

毕业答辩演示看板。纯前端单文件 Web 应用（`index.html`），内联 JS/CSS，无构建步骤，无后端依赖。

## 技术栈

- 单文件：`index.html`（HTML + 内联 CSS + 内联 JS）
- 零依赖，不引入任何 npm 包或 CDN 脚本
- 数据全部为模拟数据，与 DDoS-Test 论文实验数值对齐

## 演示数据基准（不可更改）

| 场景 | 关键指标 |
|------|---------|
| 基线 | 总 pps ≈ 120，流量 0.075 Mbps，合法成功率 100% |
| UDP Flood 无防御 | 均值 300,000 pps，峰值 400,000，合法成功率 4% |
| SYN Flood 无防御 | 均值 163,000 pps，合法握手成功率 2% |
| UDP Flood 有防御 | 拦截率 99.5%，合法吞吐恢复至 97%，CUSUM 告警 3.2 s |
| SYN Flood 有防御 | 拦截率 99.2%，合法吞吐恢复至 94%，CUSUM 告警 4.5 s |

## 开发

直接用浏览器打开 `index.html`，无需任何构建或服务器。

```bash
# 本地快速预览（WSL）
explorer.exe index.html
# 或
python3 -m http.server 8080
```

## 注意事项

- 所有修改都在 `index.html` 这一个文件中完成
- 保持零依赖原则，不引入外部脚本或样式表
- 模拟数据动画的时间轴必须与论文场景描述严格对应
- 颜色方案使用 CSS 变量（`:root` 中定义），修改主题只改变量
